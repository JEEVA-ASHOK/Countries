# 🌍 Countries Explorer

A modern React application that allows users to explore countries around the world with detailed information, search functionality, and region-based filtering.

---

## 📸 Preview

![Countries Explorer Banner](https://images.unsplash.com/photo-1526778548025-fa2f459cd5ce?w=1200)

---

## ✨ Features

- 🌍 Browse all countries
- 🔍 Search countries by name
- 🌎 Filter countries by region
- 🏳️ View country flags
- 🏛️ Capital city information
- 👥 Population details
- 🗣️ Languages spoken
- 💱 Currency information
- 📱 Fully Responsive Design
- ⚡ Fast and user-friendly interface

---

## 🛠️ Built With

- React.js
- React Router DOM
- Axios
- CSS3
- REST Countries API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/countries-explorer.git
```

### Navigate to project

```bash
cd countries-explorer
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm start
```

or

```bash
npm run dev
```

---

## 🌐 API Used

This project uses the **REST Countries API** to fetch country information.

Features include:

- Country Name
- Flag
- Capital
- Population
- Region
- Subregion
- Languages
- Currency
